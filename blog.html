Question 1: What are the different ways to select an element in the DOM?

There are several ways to select elements in the DOM:

  • getElementById("id") - Selects an element by its ID.
  • getElementsByClassName("class") - Selects elements by class (HTMLCollection).
  • getElementsByTagName("tag") - Selects elements by tag name.
  • querySelector("selector") - Selects the first matching CSS selector.
  • querySelectorAll("selector") - Selects all elements matching a CSS selector.

Question 2: What is the difference between innerHTML, innerText, and textContent?

innerHTML: Returns or sets HTML content inside an element.
innerText: Returns only visible text (ignores hidden elements).
textContent: Returns all text inside an element, including hidden text but without parsing HTML.

Question 3: What is event delegation in the DOM?

Event delegation is a technique where a parent element listens for events on child elements instead of attaching event listeners to each child individually. This is useful for dynamically added elements.

Question 4: What is event bubbling in the DOM?

Event bubbling is when an event starts from the target element and propagates up to parent elements.

Question 5: How do you create, add, and remove elements using JavaScript?

- Create: `document.createElement("tag")`
- Add: `parentElement.appendChild(newElement)`
- Remove: `element.remove()`
